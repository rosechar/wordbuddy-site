# wordbuddy-site

Public static site for the **Word Buddy** iOS app — currently just the privacy
policy required for the App Store listing.

- Live privacy policy: `privacy.html` (served via GitHub Pages)
- `index.html` redirects to the privacy policy.

This repo is intentionally public so GitHub Pages (free tier) can serve it. It
contains **no app source code** — that lives in a separate private repo.
