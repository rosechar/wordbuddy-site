# wordbuddy-site

Public static site for the **Word Buddy** iOS app — the privacy policy and
support page required for the App Store listing.

- Privacy policy: `privacy.html` (served via GitHub Pages)
- Support page: `support.html`
- `index.html` redirects to the privacy policy.

This repo is intentionally public so GitHub Pages (free tier) can serve it. It
contains **no app source code** — that lives in a separate private repo.
